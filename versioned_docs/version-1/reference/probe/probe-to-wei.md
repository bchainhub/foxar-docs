### NAME

probe-to-wei - Convert an eth amount to wei.

### SYNOPSIS

`probe to-wei` [*options*] _value_ [*unit*]

### DESCRIPTION

Convert an eth amount to wei.

Consider using [`probe to-unit`](./probe-to-unit.md).

#### Common Options

`-h`  
`--help`  
&nbsp;&nbsp;&nbsp;&nbsp;Prints help information.
