### NAME

probe-from-wei - Convert wei into an ETH amount.

### SYNOPSIS

`probe from-wei` [*options*] _value_ [*unit*]

### DESCRIPTION

Convert wei into an ETH amount.

Consider using [`probe to-unit`](./probe-to-unit.md).

### OPTIONS

#### Common Options

`-h`  
`--help`  
Prints help information.
