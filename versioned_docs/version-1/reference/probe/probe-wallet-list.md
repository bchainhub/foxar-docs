### NAME

probe-wallet-list - List all the accounts in the keystore default directory.

### SYNOPSIS

`probe wallet list`

### DESCRIPTION

List all the accounts in the keystore default directory `~/.foxar/keystores`.

#### Common Options

`-h`  
`--help`
