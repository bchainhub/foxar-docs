### NAME

probe-from-utf8 - Convert UTF8 text to hex.

### SYNOPSIS

`probe from-utf8` [*options*] *text*

### DESCRIPTION

Convert UTF8 text to hex.

### OPTIONS

#### Common Options

`-h`  
`--help`  
&nbsp;&nbsp;&nbsp;&nbsp;Prints help information.

### EXAMPLES

1. Convert UTF8 text to hex:
   ```sh
   probe from-utf8 "hello"
   ```
